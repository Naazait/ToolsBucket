About Color Models

  • HEX (Hexadecimal): Commonly used in web design, it represents colors with a six-digit code (e.g., `#007BFF`).
  • RGB (Red, Green, Blue): Used for digital displays like monitors and screens. It mixes red, green, and blue light (0-255 for each).
  • HSL (Hue, Saturation, Lightness): An intuitive model where Hue is the color angle (0-360), and Saturation/Lightness are percentages.
  • CMYK (Cyan, Magenta, Yellow, Key/Black): A subtractive model used in color printing.
