Frequently Asked Questions (FAQs)

Is it secure to use this tool?

Yes, this tool is completely secure. All operations (locking and unlocking) happen directly in your web browser. Your PDF files are never uploaded to our servers, ensuring your data remains private. We prioritize your privacy above all else.

Can this tool remove a password if I have forgotten it?

No. For security reasons, you must know the current password to unlock a PDF. This tool cannot bypass or 'crack' a password you do not know. It's designed for legitimate users who own the documents and their respective passwords.

What kind of PDFs can I process?

Our tool works with standard PDF documents. As long as the PDF is not corrupted, you should be able to lock or unlock it successfully.

Is there a limit to file size or number of uses?

Currently, there are no strict limits on file size or the number of times you can use the tool. However, very large files might take slightly longer to process depending on your computer's performance.

What happens if I try to lock an already locked PDF?

If you attempt to lock an already password-protected PDF, the tool will inform you that the PDF is already locked. To apply a new password, you would first need to unlock it using the current password.

What happens if I try to unlock an unprotected PDF?

If you try to unlock a PDF that is not password-protected, the tool will notify you that the document is not encrypted.
